# APECI S.A. — Portal de Exportación Global
 
Sitio web simulado para la empresa **APECI S.A.** (Asociación de Productores de Espárragos Frescos y en Conserva), desarrollado como proyecto académico de negociación internacional.
 
## Descripción
 
Portal B2B de exportación de espárragos frescos desde Perú hacia el mercado europeo (Alemania). Incluye ficha técnica del producto, condiciones logísticas, certificaciones internacionales, análisis de costos y formulario RFQ.
 
## Estructura de archivos
 
```
apeci/
├── index.html    # Estructura principal con navegación por secciones
├── style.css     # Estilos responsive y diseño premium
├── script.js     # Lógica de navegación, gráfico Chart.js y formulario
└── README.md     # Este archivo
```
